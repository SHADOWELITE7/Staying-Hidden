# Staying Hidden

Staying Hidden is a Stealth Based Roguelike

The project theme takes inspiration from [Quake](https://en.wikipedia.org/wiki/Quake_(series))'s Dark Atmosphere and [Thief](https://en.wikipedia.org/wiki/Thief_(series))'s revolutionary stealth gameplay. 

There is no deadline to this project as I rather take my time making this as a passion project
